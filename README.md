# About

I build **local-first AI systems** designed to compose **autonomous multi-agent architectures**.

Each repository in this profile represents a **modular skill** that can be combined with others to form **coordinated SuperAgents capable of real-world automation**.

The focus is simple:

• local execution  
• privacy by design  
• open systems  
• no cloud lock-in
