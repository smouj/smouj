# 🧠 About

I design **modular ecosystems of AI skills** capable of composing autonomous agents.

These systems run **local-first**, prioritize **privacy**, and avoid **cloud lock-in**.

Each skill acts as a building block that can combine into **coordinated SuperAgents** capable of real-world automation.
