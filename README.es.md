# 🧠 Sobre mí

Construyo **ecosistemas modulares de skills de IA** capaces de formar **agentes autónomos**.

Todos los sistemas siguen tres principios:

- **Local-first**
- **Privacidad por diseño**
- **Código abierto**

Cada skill es una pieza.  
Juntas forman **SuperAgentes capaces de automatizar tareas reales**.
